# BPO Operations Dashboard

This repository contains a **comprehensive Excel dashboard** for BPO operations.  
It is designed to track, visualize, and analyze key performance metrics, manpower planning, quality, and attrition data for call center operations.

---

## **Dashboard Overview**

The dashboard provides a complete view of BPO operations with the following modules:

### **1. Manpower Planning**
- Total agents available and scheduled
- Planned vs. actual shifts
- Occupancy and adherence tracking

### **2. KPI Cards**
- **Total Calls** handled
- **Average Handling Time (AHT)**
- **Customer Satisfaction (CSAT)**
- **SLA Compliance**
- **Adherence Rate**

### **3. Quality Analysis**
- QA scores and trend analysis
- Identification of areas needing improvement

### **4. Attrition Tracking**
- Monthly attrition percentage
- Reasons for employee turnover
- Team-wise and agent-wise attrition insights

### **5. Agent Performance**
- Individual agent metrics:
  - Calls handled
  - AHT
  - CSAT
  - Quality scores
- Enables comparison across agents and teams

---

## **How to Use the Dashboard**

1. Open the Excel file: `BPO_Operations_Dashboard.xlsx`.
2. Navigate to the **Dashboard** sheet to view all KPIs, charts, and insights.
3. To refresh the dashboard:
   - Update the raw data sheets:
     - **Manpower Planning**
     - **KPI Summary**
     - **Quality Analysis**
     - **Attrition**
     - **Agent Performance**
   - The charts and KPI cards will automatically update.
4. Use slicers (if included) to filter data by **Month, Team, or Agent** for detailed analysis.
